# Signature Tool

A simple tool where you can draw your signature and download it. That's it.

I built this with just HTML, CSS and JavaScript — no libraries or anything fancy.

---

## What it does

- Draw your signature with mouse or finger (works on phone too)
- Pick from 3 colors — black, red, blue
- Make the pen thicker or thinner with a slider
- Download as PNG or SVG
- Clear button if you mess up and want to redo it

---

## How to use it

Just download the `index.html` file and open it in your browser. No setup needed, no install, nothing.

1. Draw your signature on the canvas
2. Pick a color and pen size if you want
3. Choose PNG or SVG
4. Hit Download

The download button won't work until you actually draw something, which makes sense.

---

## PNG or SVG?

Honestly just use PNG, it works everywhere. SVG is if you need to resize it without it getting blurry.

---

## Why I made this

I wanted a clean simple signature tool without all the extra stuff most online tools have. No sign up, no watermark, no ads. Just draw and download.
